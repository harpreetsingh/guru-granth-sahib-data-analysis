# Project Instructions

## Blog Writing Rules

- When referencing data, statistics, or analysis findings in blog posts, always link to the relevant detailed analysis document in `docs/` using the GitHub URL format: `https://github.com/harpreetsingh/guru-granth-sahib-data-analysis/blob/main/docs/{filename}`
- The mapping of topics to docs:
  - Corpus stats, languages, ragas, authorship: `01-corpus.md`
  - Entity matching, frequency tables: `02-matching.md`
  - Feature dimensions, lexicon: `03-features.md`
  - Density analysis: `04-density.md`
  - Co-occurrence, PMI: `05-cooccurrence.md`
  - Tagging, register classification: `06-tagging.md`
  - Register mixing: `07-mixing.md`
  - Per-author profiles: `08-authors.md`
  - Semantic analysis (RAM, ALLAH, civilizational): `09-semantic.md`
  - Key findings summary: `10-findings.md`
  - Glossary of terms: `glossary.md`
