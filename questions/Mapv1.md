# MASTER QUESTION MAP

GGS Multi-Layer Textual Analysis Framework

## I. LEXICAL LAYER (Pure Token Reality — No Interpretation)

### A. Divine Name Frequency

**Core Divine Tokens**

- ਰਾਮ (Ram)
- ਹਰਿ (Hari)
- ਗੋਵਿੰਦ (Govind)
- ਮਾਧਵ (Madhav)
- ਕ੍ਰਿਸ਼ਨ / ਕਿਸਨ (Krishan/Kishan)
- ਵਿਸ਼ਨੁ (Vishnu)
- ਨਾਰਾਇਣ (Narayana)
- ਪ੍ਰਭੁ (Prabhu / Lord)
- ਸਾਹਿਬ (Sahib / Master)
- ਬ੍ਰਹਮ (Brahm / Absolute)

**Islamic divine markers:**

- ਅਲਾਹੁ (Allah)
- ਖੁਦਾ (Khuda)
- ਰਹਮਾਨ (Rahman)
- ਰਹੀਮ (Rahim)

**Questions**

- How many times does each token appear?
- What % of total lines contain at least one divine-name token?
- Distribution across:
  - Ang
  - Author (Mahalla)
  - Raga

### B. Mythological Narrative Markers (Strong Sagun Signals)

**Epic-specific tokens:**

- ਦਸ਼ਰਥ (Dashrath)
- ਸੀਤਾ (Sita)
- ਲਖਮਣ (Lakshman)
- ਅਯੋਧਿਆ (Ayodhya)
- ਕੰਸ (Kansa)
- ਗੋਪੀ (Gopi)
- ਇੰਦਰ (Indra)
- ਬ੍ਰਹਮਾ (Brahma -- deity, not Brahm abstract)

**Questions:**

- How many explicit epic references?
- What % of divine references are explicitly narrative?
- Which authors use them?

### C. Scriptural References

**Indic scriptural markers:**

- ਵੇਦ (Vedas)
- ਪੁਰਾਣ (Puranas)
- ਸ਼ਾਸਤਰ (Shastras)

**Islamic scriptural markers:**

- ਕੁਰਾਨ / ਕੁਰਾਣ (Qur'an)
- ਕਿਤਾਬ (Kitab / Book)
- ਨਬੀ (Prophet)
- ਪੈਗੰਬਰ (Messenger)

**Questions:**

- Total count per category
- Ratio Indic : Islamic
- Context polarity (neutral / critique / endorsement)

### D. Religious Office / Clerical Markers

**Indic:**

- ਬ੍ਰਾਹਮਣ (Brahmin)
- ਪੰਡਿਤ (Pandit)
- ਜੋਗੀ (Yogi)
- ਸਿੱਧ (Siddh)

**Islamic:**

- ਮੁਲਾਂ (Mullah)
- ਕਾਜ਼ੀ (Qazi)
- ਫਕੀਰ (Faqir)

**Questions:**

- Frequency
- Critique vs neutral
- Author clustering

### E. Ritual Markers

**Indic ritual:**

- ਯਗ / ਯਜ੍ਨ (Yagya / Sacrifice)
- ਹਵਨ (Havan)
- ਮੂਰਤਿ (Idol)
- ਤੀਰਥ (Pilgrimage site)

**Islamic ritual:**

- ਨਮਾਜ਼ (Namaz / Prayer)
- ਰੋਜ਼ਾ (Roza / Fast)
- ਹੱਜ (Hajj)
- ਮਸਜਿਦ (Masjid / Mosque)

**Questions:**

- Density
- Co-occurrence with negation
- Distribution by Guru

## II. STRUCTURAL LAYER

### A. Author Gravitation

For each Guru / Bhagat:

- Divine token density
- Nirgun marker density
- Sagun narrative density
- Perso-Arabic vocabulary density
- Sanskritic vocabulary density

**Questions:**

- Do Gurus differ statistically?
- Are Bhagat compositions lexically distinct?

### B. Ang Progression

- Do divine tokens cluster in specific Ang ranges?
- Does Islamic vocabulary spike in specific sections?
- Is Nirgun density stable or uneven?

### C. Raga Correlation

Do certain Ragas correlate with:

- Nirgun markers?
- Mythological markers?
- Perso-Arabic vocabulary?

### D. Co-occurrence Graph

**Examples:**

- ਰਾਮ + ਅਲਾਹੁ (Ram + Allah in same line)
- ਹਰਿ + ਗੋਵਿੰਦ (Hari + Govind)
- ਬ੍ਰਾਹਮਣ + ਮੁਲਾਂ (Brahmin + Mullah)

**Questions:**

- Frequency of cross-tradition pairing
- Collapse-of-duality density

## III. INTERPRETIVE LAYER

### A. Nirgun Markers (Formless)

- ਨਿਰਗੁਣ (Nirgun)
- ਨਿਰੰਕਾਰ (Nirankar)
- ਅਲਖ (Unseen)
- ਅਗਮ (Inaccessible)
- ਅਗੋਚਰ (Beyond senses)
- ਅਕਾਲ (Timeless)
- ਅਜੂਨੀ (Unborn)
- ਸੁੰਨ (Void)
- ਅਨਹਦ (Unstruck sound)
- ਨਾਮ (Naam -- contextual)

**Questions:**

- % of lines containing Nirgun markers
- Density by Guru
- Co-occurrence with divine names

### B. Sagun Markers (With Form)

**Strong signals:**

- Ramchandra-style narrative markers
- Krishna narrative markers
- Explicit deity names in myth context

**Questions:**

- % lines with explicit Sagun narrative
- Pure Sagun vs mixed vs pure Nirgun lines

### C. Critique vs Affirmation

Lines referencing:

- Vedas
- Quran
- Ritual practice
- Clerical authority

Classify as:

- critique
- neutral
- transcendence
- affirmation

### D. Universalist Collapse Markers

Lines containing:

- Hindu + Muslim markers together
- Caste negation
- Equality language
- "No Hindu, no Muslim" type constructions

Measure density.
